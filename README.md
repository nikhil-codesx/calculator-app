# 🚀 Premium Calculator App

A modern **3D animated calculator** built using HTML, CSS, and JavaScript.
Designed with a **dark violet premium UI**, smooth animations, and interactive effects.

---

## ✨ Features

* 💜 Dark premium UI (glassmorphism design)
* ⚡ Smooth hover & click animations
* 🧊 3D tilt interaction (mouse movement)
* ⌨️ Keyboard support
* 🔥 Neon glow effects
* 📱 Responsive design

---

## 🛠️ Tech Stack

* HTML5
* CSS3
* JavaScript
